## Catatan Struktur Project

Project ini menggunakan Vite sebagai development server, sehingga struktur
folder sedikit berbeda dari contoh soal, namun fungsionalitas dan arsitektur
tetap sama.

Mapping struktur:

- folder `scripts/` setara dengan `js/` pada soal
- folder `styles/` setara dengan `style/` pada soal
- file di `scripts/pages/` merupakan implementasi halaman SPA
